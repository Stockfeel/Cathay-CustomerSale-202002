# cathay-customer

## Tech stack

本專案主要使用 `vue-router`, `styled-component` 開發，
細節、版本部份請參照 `package.json`
Node 版本使用最新 stable 版本即可

## 目錄說明 

`
/src 
/src/assets 圖片
/src/assets/components/ 各頁切版檔案 
/src/fonts 字型檔
/src/router 路徑
/src/style.js style 元件
`

## 主要 npm scripts
```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

```
npm run build
```
